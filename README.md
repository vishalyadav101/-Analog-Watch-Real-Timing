# ⌚ Analog Watch – Premium UI

A beautifully designed **Analog Watch** built using **HTML, CSS, and JavaScript**.
This project features a real-time working clock with smooth hand movements, Roman numerals, and an elegant premium UI.

---

## 🚀 Features

* 🕒 Real-time Analog Clock (Hour, Minute, Second hands)
* ⏱ Smooth animation using `requestAnimationFrame`
* 📅 Live Date display (Day, Month, Year)
* 🔢 Roman Numeral Dial
* 💎 Premium luxury watch UI design
* 📱 Fully centered and responsive layout
* 🎨 Custom CSS styling with gradients and effects

---

## 🛠️ Technologies Used

* HTML5
* CSS3 (Advanced styling, gradients, animations)
* JavaScript (DOM manipulation, real-time updates)
